What I Do


Responsive Design

I build with a responsive approach. Site content should be able to adapt to any container, whether that's a phone or a desktop computer. This is increasingly important as mobile browsing continues to grow.


Cross-Browser Compatibility

Chrome, Safari, Firefox, Internet Explorer. There are a lot of different browsers out there and I aim to have a beautiful, functioning site, no matter which browser is used.


Best Practices

Technology is constantly changing, so I make sure to stay in-tune with the latest practices and techniques. Clean, semantic and stable code is what I push for.
